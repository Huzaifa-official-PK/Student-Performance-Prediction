Machine Learning Models

Evaluated regression models and key strengths — final selection highlighted below.

Linear Regression (Final Model)

A simple, interpretable regression model used as the final production model for consistent performance and explainability.

  • Highly interpretable coefficients
  • Fast training and prediction
  • Robust with regularization
  • Minimal overfitting on tabular data
Random Forest Regressor

Ensemble of decision trees offering strong performance on non-linear relationships and robust averaging.

  • Captures non-linear patterns
  • Resistant to overfitting via averaging
  • Handles mixed feature types well
Gradient Boosting Regressor

Boosting-based ensemble that builds sequential trees to correct previous errors, often producing high accuracy.

  • High predictive power
  • Effective with careful tuning
  • Handles complex feature interactions
Extra Trees Regressor

Extremely randomized trees provide faster training and often strong accuracy by reducing variance.

  • Very fast training
  • Lower variance through randomization
  • Good default for tabular data
XGBoost Regressor

Optimized gradient boosting implementation delivering excellent accuracy and flexibility with regularization.

  • State-of-the-art gradient boosting
  • Built-in regularization
  • Highly tunable for performance

Project Overview

This project predicts a student's expected exam score using Machine Learning. It analyzes academic, personal, family and lifestyle factors to estimate final performance.

The application is developed using Flask, Scikit-Learn, Bootstrap 5 and Python with a responsive modern dashboard.
